# Slike za sekcijo "Izbrana dela"

V to mapo dodajte slike, ki naj bodo prikazane na domači strani v sekciji **Predstavitev / Izbrana dela**.

Priporočila:

- Uporabite formate `.jpg`, `.jpeg`, `.png` ali `.webp`.
- Imena datotek naj bodo brez presledkov in šumnikov, npr. `lesena-gravura.jpg` ali `k9-kristal.webp`.
- Priporočena vodoravna oblika slike je približno razmerje 4:3.
- Ko dodate slike, bo treba v `index.html` zamenjati obstoječe povezave slik z lokalnimi potmi, npr. `assets/gallery/lesena-gravura.jpg`.
